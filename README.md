- 而在这江湖之中，我又将成为什么？是大侠、是富豪、是一派宗师？还是狗屎、虾米、无名小卒？我不知道，但你能！

## 合约介绍
### contract.jhqsj
- 提供玩家操作的接口
- 注册
- 上吊(清空所有私有数据)
- 农场
### contract.jh-player
- 管理玩家属性
### contract.jh-package
- 管理玩家背包
### contract.jh-farms
- 农场模块
- 农场奸商,种子\铁锹购买
- 土地购买\升级
- 收割和种植
### contract.jh-itemfactory
- 管理物品属性生成概率
- 管理物品配置
### contract.jh-gift
- 管理暗号和对应的礼包
